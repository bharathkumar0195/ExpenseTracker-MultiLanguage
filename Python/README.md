# Expense Tracker – Python Implementation

## Project Overview
This folder contains the Python implementation of the Expense Tracker application developed as part of the Cross-Language Application Development project.

The application allows users to manage and analyze daily expenses through a console-based interface.

---

## Features Implemented

- Add new expense entries
- Predefined expense categories
- Date-based expense filtering
- Category-based filtering
- Summary report with total calculations
- Input validation and error handling
- Modular function-based structure

---

## Technologies Used

- Python 3.x
- Console-based user interaction
